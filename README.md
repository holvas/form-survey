Приклад форми для опитування

ТЗ:
Титул сторінки в елементі h1 з class зі значенням title
Коротке роз'яснення в елементі p з class зі значенням description
Елемент form з class зі значенням survey-form
В елементі форми - потрібно ввести ім'я в полі input, яке має class зі значенням name та type зі значенням text
В елементі форми - потрібно ввести електронну пошту в полі input, яке має class зі значенням email
Якщо ввести неправильно сформовану електронну адресу, то побачите помилку перевірки HTML5
В формі можна ввести число в поле input, яке має class зі значенням number
Ввід для чисел повинен приймати лише числа, або забороняючи вводити їх, або показуючи помилку перевірки HTML5 (залежно від вашого браузера).
Якщо ввести число поза межами діапазону чисел, що визначено атрибутами min та max, то побачите помилку перевірки HTML5
Для імені, електронної пошти та полів введення чисел можна побачити відповідні елементи label в формі, що описують мету кожного поля відповідними ідентифікаторами: class="name-label", class="email-label" та class="number-label"
Для імені, електронної пошти та полів введення чисел можна побачити текст заповнювача з описом або інструкцією до кожного поля
Елементи форми повинні мати випадний елемент select з class зі значенням dropdown та принаймні два варіанти вибору
Елементи форми повині мати до вибору опцію з групи принаймні двох радіокнопок, які згруповані з використанням атрибуту name
Елемент форми може вибрати декілька полів з низки прапорців, кожен з яких повинен мати атрибут value
В елементі форми представлено textarea для додаткових коментарів
В елементі форми представлено кнопку з class зі значенням submit для відправки всіх введень
Оформити за власним стилем.
